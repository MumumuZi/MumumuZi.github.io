佩奇排名(PageRank)

发表于 2018-07-25

算法背景

  最早的搜索引擎采用的是分类目录的方法，即通过人工进行网页分类并整理出高质量的网站。后来网页越来越多，人工分类已经不现实了。搜索引擎进入了文本检索的时代，即计算用户查询的关键字与网页内容的相关程度来返回搜索结果。但是搜索效果不是很好，因为总有某些网页来回使用某些关键词使自己的搜索排名靠前。谷歌的两位创始人，当时还是斯坦福大学研究生的佩奇（Larry Page）和布林（Sergry Brin）开始了对网页排序问题的研究。他们借鉴了学术界评判学术论文重要性的方法——看论文的引用次数，由此想到网页的重要性也可以利用这种方法，于是产生了PageRank的核心思想。
  佩奇排名(PageRank)，又称网页排名、谷歌左侧排名，它通过网络浩瀚的超链接关系来确定一个页面的重要性。Google把从A页面到B页面的链接解释为A页面给B页面投票，Google根据投票来源（甚至来源的来源，即链接到A页面的页面）和投票目标的等级来决定新的等级。

基本原理

  PageRank算法总的来说就是预先给每个网页一个PR得分，PR值的意义是网页被访问概率，所以一般是1/N，其中N为网页总数。

例如上图中，有两条边指向A，其中B除了指向A还指向了D，所以A的PR值的计算方法为：

PR(A)=PR(C)+PR(B)/2

当遇到没有出链的网页时，如下图所示，


其中C没有出链，没有为其他网页的PR值做贡献，所以假设它对所有网页（包括它自己）都有出链，因此图中A的PR值为：
PR(A) = PR(B)/2 + PR(C)/4

再考虑一种情况，某个网页只有对自己的出链，或者几个网页的出链形成一个环，那么在不断的迭代过程中，这些网页的PR值会一直增加，显然这不合理，所以我们假定浏览网页的人有一个确定的概率会直接跳转到一个随机的网页，并且跳转到每个网页的概率都是一样的，则下图中PR（A）的算法为：


PR(A) = PR(B)/2 * d + (1 - d)/N

其中d即为阻尼系数，为0.85。其意义是，在任意时刻，用户到达某页面后并继续向后浏览的概率，该数值是根据上网者使用浏览器书签的平均频率估算而得。1-d=0.15，即用户停止点击，随机跳到新URL的概率。
