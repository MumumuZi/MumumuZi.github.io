AFSA学习笔记

发表于 2018-07-31

主要记录了在学习人工鱼群算法时的一些笔记

一些定义

  • 人工鱼群算法（AFSA）：是由我国学者李晓磊博士在2003年首次提出的一种仿生优化算法，是一种基于动物行为寻求全局最优解的新思路，是行为主义人工智能的一个典型应用。其基本思想是在一片水域中，鱼能自行或尾随其他鱼找到营养位置多的地方，因而鱼生存数量最多的地方一般就是本水域中营养物质最多的地方。人工鱼群算法则通过模拟鱼群的各种行为和动作来优化要解决的问题。
  • 鲁棒性：指系统在扰动或不确定的情况下仍能保持它们的特征行为。

算法原理

  人工鱼群算法四个算子：自由游动、觅食操作、追尾操作、聚群操作。鱼群中的个体所处的状态即为待求解向量，算法中通常设置一个公告板来记录鱼群中当前最优个体状态。

算法的主要参数

  • Visual：鱼的视野范围——当视野范围较小时，人工鱼的觅食行为和随机行为比较突出；视野范围较大时，人工鱼的追尾行为和聚群行为比较突出。总的来说：视野越大，越容易使人工鱼发现全局最优解并收敛。
  • λ：拥挤因子——在求极大值问题中，λ=1/(αNmax)，α∈（0,1]；在求极小值问题中，λ=αNmax，α∈(0,1]。其中α为极值接近水平，Nmax为期望在该邻域内聚集的最大人工鱼数目
  • step：移动步长——对于固定步长，随着步长的增加，收敛速度得到了一定的加速，但在超过一定的范围后，又使得收敛速度减缓，步长过大时会出现震荡现象而大大影响收敛速度。采用随机步长的方式在一定程度上防止了震荡现象的发生，并使得该参数的敏感度大大降低了，但最快的收敛速度还是最优固定步长的收敛速度。步长越小，求解精度越高。
  • try_number：尝试次数——尝试次数越多，人工鱼的觅食行为能力越强，收敛的效率也越高。在局部极值突出的情况下，应该适当的减少以增加人工鱼随机游动的概率，克服局部最优解。
  • fish_number：鱼群数量
  • itermax：最大迭代次数
  • dim：维度

人工鱼四种基本行为

  将当前鱼个体的状态表示为X=（X1，X2，…Xn），所处位置的食物浓度设为F(X)，即求解目标函数。
自由游动：设鱼的当前状态为Xi，在没有执行其他任何行为的时候，鱼个体在自己视野范围Visual内随机游动。
觅食行为：设鱼的当前状态为Xi，在其视野范围Visual内随机选择一新状态Xj,如果F(xj) > F(xi)(假设最小优化问题)，则鱼向该状态移动一步；否则，重新选择一个新的Xj，进行尝试。尝试try_number次后仍然不能移动，则将鱼随机移动一步。
聚群行为：设鱼的当前状态为Xi，在其视野范围Visual内搜索聚集鱼群的中心位置Xc，并探测附近的同伴个数S。如果S/N < λ，表示该中心位置不拥挤，如果此时F(Xc) < F(Xi),则鱼向该方向前进一步，否则执行觅食行为。
追尾行为：设鱼的当前状态为 Xi，在其视野范围内搜索最优的鱼个体Xmin。设 XMIN 视野领域内的伙伴数为S’，如果 F(Xi)<F(Xmin)并且S’/ N < λ,表明该位置食物较多，且不拥挤，则当前鱼向该位置前进一步，否则继续觅食行为。
更新公告板：设一个公告板，用于记录鱼群内历史最佳鱼的状态Xbest。各人工鱼每迭代一次都检查自身状态，如果 F(Xi)<F(Best)，则将 Xbest修改为 Xi。

寻优原理

  人工鱼群算法在寻优的过程中，可能会集结在几个局部最优解的周围，使人工鱼跳出局部最优解，实现全局寻优的因素主要有：

  • 觅食行为中重复次数较少时，为人工鱼提供了随机移动的机会，从而可能跳出局部最优解；
  • 随机步长使得人工鱼在前往局部最优解的途中，有可能转向全局最优解；
  • 拥挤度因子λ限制了聚群的规模，使得人工鱼能够更广泛的寻优；
  • 聚群行为能够促使陷于局部最优解的人工鱼趋向全局最优解的人工鱼方向聚集，从而逃出局部最优解；
  • 追尾行为加快了人工鱼向更优状态游动。
