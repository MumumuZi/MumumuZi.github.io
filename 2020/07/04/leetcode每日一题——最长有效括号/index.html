leetcode每日一题——最长有效括号

发表于 2020-07-04

题目描述

Given a string containing just the characters ‘(‘ and ‘)’, find the length of the longest valid (well-formed) parentheses substring.

给定一个只包含 '(' 和 ')' 的字符串，找出最长的包含有效括号的子串的长度。

方法一：栈

看到这道题的第一直觉就是用栈，输出的是长度，所以考虑将索引入栈，方便计算长度，大概思路如下：

  • 遇到左括号则将下标入栈，等待被匹配。

  • 遇到右括号，先出栈表示匹配成功（会存在第一个符号就是右括号的情况，所以先入栈一个-1作为栈底元素）。
    如果栈为空，说明将栈底元素-1弹出了，该右括号其实没有被匹配，将其入栈作为新的栈底元素

    如果栈不为空，当前右括号的下标减去栈顶元素即为以该右括号为结尾的有效括号的长度，与最长有效括号长度比较进行更新即可。图解如下：图片出处

    例如，当遇见索引5的右括号时，栈顶元素2出栈，栈顶变为-1，此时的有效长度为5-（-1）=6，大于初始长度0，更新当前最大长度为6。

    当遇到索引6的右括号时，栈顶元素-1出栈，此时栈为空，则6入栈为新的栈顶元素，这样当遇到8右括号时，长度为8-6=2<6，不更新最大长度。
